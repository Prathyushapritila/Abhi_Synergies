# Abhi Elite Services - Marketing Website

A complete, production-quality, fully responsive marketing website for Abhi Elite Services built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Tech Stack

- **Next.js 14** with App Router
- **TypeScript**
- **Tailwind CSS**
- **React** functional components

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Contact form with client-side validation
- Accessible components with proper ARIA labels
- Clean, premium corporate design
- Custom color scheme and typography

## Build for Production

